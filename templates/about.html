About AMLPDS

Project Overview

The Adversarial Machine Learning for Phishing Detection System (AMLPDS) is an advanced cybersecurity solution that combines state-of-the-art natural language processing with adversarial training techniques to detect sophisticated phishing attempts.

Traditional phishing detection systems often fail against carefully crafted attacks. AMLPDS addresses this vulnerability by training on adversarially generated examples, making it robust against attempts to evade detection.

Technology Stack

  • Model: BERT (Bidirectional Encoder Representations from Transformers)
  • Framework: PyTorch & Hugging Face Transformers
  • Adversarial: TextAttack with TextFooler
  • Explainability: SHAP (SHapley Additive exPlanations)
  • Backend: Flask with JWT authentication
  • Frontend: Tailwind CSS & JavaScript

Performance Metrics

  • Accuracy: 94.38%
  • F1-Score: 94.9%
  • Precision: 94.2%
  • Recall: 95.6%
  • Robust Accuracy: 88.2% (against adversarial attacks)
  • Average Response Time: 0.422 seconds
